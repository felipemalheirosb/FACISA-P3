Problema do Diamante

A Linguagem Java implementa apenas herança simples, a subclasse
precisa ter umas relação 'é um' com todas as sua superclasses.
A linguagem java não permite que uma subclasse herde de duas
superclasses ao mesmo tempo, pois não saberia quais caracteristicas
iria herdar.

      Classe A
       /   \
      /     \
Classe B    Classe C
     \      /
      \    /
     Classe D
